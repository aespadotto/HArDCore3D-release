A class representing a vertex of a cell for a 3D mesh. Contains a pointer to the mesh, the vertex coordinates, and cells, faces, edges and vertices linked to that vertex. Constructor. Vertices are constructed first, we thus cannot provide the list of cells, faces, edges they are linked to, this has to be filled in using "add_..." after the edges, faces and cells are constructed
